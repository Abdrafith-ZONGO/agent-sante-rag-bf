# G6-Assistant-Juridique-Burkina

Agent IA Assistant & Système RAG pour l'aide aux démarches administratives et juridiques au Burkina Faso.

Projet Data Science - Master 1 IFOAD (Édition 2026)

## Objectif

Concevoir un assistant intelligent capable de répondre aux questions sur les démarches citoyennes au Burkina Faso :
- CNIB
- Passeport
- Création d'entreprise
- Autres formalités administratives

L'agent utilise le RAG (Retrieval-Augmented Generation) pour interroger une base de connaissances locale et fournir des réponses précises basées sur des textes officiels.
